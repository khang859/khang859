A cross-platform desktop app for developers running multiple AI coding agents at once. One window to manage all your terminal sessions, with features built around AI-assisted workflows.

- **AI agent monitoring** — real-time activity tracking, permission request notifications, and a Copilot overlay across all sessions
- **Splits, tabs, and workspaces** — drag-and-drop layout with keyboard navigation
- **Built-in tooling** — file editor, git integration, worktree management, and a socket API for programmatic control
